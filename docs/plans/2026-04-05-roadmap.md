## 阶段1: 数据层加固 (1-2周)

### 1.1 多数据源抽象

**目标**: 封装 Finnhub/Polygon 作为 Yahoo Finance 的替代，数据源可切换

```
src/lib/data/
├── data-source.ts          # 已有，改造成多数据源
├── sources/
│   ├── yahoo-finance.ts   # 现有逻辑迁移

├── data-source-registry.ts # 数据源注册与切换
└── data-source-config.ts  # 配置管理
```

**数据源接口约定**:
```typescript
interface MarketDataSource {
  getHistoricalData(symbol: string, range: string): Promise<HistoricalPrice[]>
  getQuote(symbol: string): Promise<Quote>
  search(query: string): Promise<StockSearchResult[]>
  getFundamentalData(symbol: string): Promise<FundamentalData>
  healthCheck(): Promise<boolean>
}
```

**关键决策**:
- 数据源选择优先级: Yahoo (免费, 速度慢) → Finnhub (免费层 60req/min) → Polygon (付费, 最稳定)
- 配置方式: 环境变量 `PRIMARY_DATA_SOURCE=yahoo` 或 `DATA_SOURCES=yahoo,finnhub,polygon`
- 熔断策略: 单个数据源连续失败3次自动切换下一个

## 阶段3: 因子系统深化 (1-2周)

### 3.1 因子有效性看板

**目标**: 每个因子展示历史 IC、IC_IR、分组收益单调性

```
src/app/analysis/ → 因子筛选 Tab → "因子分析" 子面板
src/components/factors/
├── FactorEffectiveness.tsx    # 有效性看板
├── IcChart.tsx               # IC 时序图
├── IcIrChart.tsx             # IC_IR 柱状图
└── GroupReturnChart.tsx      # 分组收益热力图
```

**因子 IC 分析**:
```typescript
interface FactorEffectiveness {
  factorId: string
  name: string
  ic: {
    current: number
    history: { date: string; ic: number }[]
    mean: number
    std: number
  }
  icIr: number              // IC均值 / IC标准差
  groupReturns: {           // 按因子值分10组，各组收益
    group: number
    return: number
  }[]
  decayTest: {              // IC 衰减速度测试
    ic_1d: number
    ic_5d: number
    ic_20d: number
  }
}
```

**计算逻辑**:
- 选取宽基股票池（如 SPY 成分股或全市场前500）
- 计算每个因子与下期收益的 IC（Information Coefficient = Pearson相关系数）
- 按因子值排序分组，看收益是否单调递增

### 3.2 因子相关性热力图

## 附录: 关键技术决策备忘

### 数据源优先级
- Yahoo Finance: 免费，速率限制 2req/s，数据完整但不稳定
- Finnhub: 免费层 60req/min，company 数据好，K线数据一般
- Polygon.io: 付费，数据最稳定，延迟低
- **决策**: 默认 Yahoo，失败时自动降级到 Finnhub，用户可手动配置优先级

### 策略编辑器评估
- 方案A (JSON UI): 灵活，但学习者仍需理解 JSON 结构
- 方案B (拖拽 UI): 更友好，但实现复杂
- **决策**: 先实现方案A（JSON + 表单结合），后续迭代拖拽

### 回测陷阱演示数据
- 使用真实历史数据 + 预设的"陷阱参数"
- 不需要模拟数据，直接用现有的 Yahoo 数据

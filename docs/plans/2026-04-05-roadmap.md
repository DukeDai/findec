### 3.1 因子有效性看板

**目标**: 每个因子展示历史 IC、IC_IR、分组收益单调性

```
src/app/analysis/ → 因子筛选 Tab → "因子分析" 子面板
src/components/factors/
├── FactorEffectiveness.tsx    # 有效性看板
├── IcChart.tsx               # IC 时序图
├── IcIrChart.tsx             # IC_IR 柱状图
└── GroupReturnChart.tsx      # 分组收益热力图
```

**因子 IC 分析**:
```typescript
interface FactorEffectiveness {
  factorId: string
  name: string
  ic: {
    current: number
    history: { date: string; ic: number }[]
    mean: number
    std: number
  }
  icIr: number              // IC均值 / IC标准差
  groupReturns: {           // 按因子值分10组，各组收益
    group: number
    return: number
  }[]
  decayTest: {              // IC 衰减速度测试
    ic_1d: number
    ic_5d: number
    ic_20d: number
  }
}
```

**计算逻辑**:
- 选取宽基股票池（如 SPY 成分股或全市场前500）
- 计算每个因子与下期收益的 IC（Information Coefficient = Pearson相关系数）
- 按因子值排序分组，看收益是否单调递增

### 3.2 因子相关性热力图

**目标**: 可视化各因子间相关性，避免选冗余因子

```
src/components/factors/
└── FactorCorrelationHeatmap.tsx
```

**功能**:
- 计算因子之间的 Pearson 相关系数矩阵
- 热力图展示（绿色=负相关, 白色=无关, 红色=正相关）
- 选中两个因子显示散点图
- 自动提示: "建议避免同时使用 MA20 和 EMA20（相关性 > 0.99）"

### 3.3 预设策略模板

**目标**: 一键模板帮助新手快速上手

```
src/components/factors/
└── StrategyTemplates.tsx
```

**预设模板**:

| 模板名 | 因子组合 | 权重 | 适合市场 |
|--------|----------|------|----------|
| 价值投资 | PE < 15 + ROE > 15% + 负债率 < 50% | 等权重 | 熊市/震荡 |
| 动量策略 | 近1月涨幅 top20% + RSI < 70 | 等权重 | 牛市 |
| 质量策略 | ROE > 20% + 稳定现金流 + 低杠杆 | 等权重 | 长期持有 |
| 低波动 | 20日波动率 bottom30% + beta < 1 | 等权重 | 风险厌恶 |

**UI**: 模板卡片展示 + 一键应用到当前策略 + 自定义修改

## 附录: 关键技术决策备忘

### 数据源优先级
- Yahoo Finance: 免费，速率限制 2req/s，数据完整但不稳定
- Finnhub: 免费层 60req/min，company 数据好，K线数据一般
- Polygon.io: 付费，数据最稳定，延迟低
- **决策**: 默认 Yahoo，失败时自动降级到 Finnhub，用户可手动配置优先级

### 策略编辑器评估
- 方案A (JSON UI): 灵活，但学习者仍需理解 JSON 结构
- 方案B (拖拽 UI): 更友好，但实现复杂
- **决策**: 先实现方案A（JSON + 表单结合），后续迭代拖拽

### 回测陷阱演示数据
- 使用真实历史数据 + 预设的"陷阱参数"
- 不需要模拟数据，直接用现有的 Yahoo 数据
